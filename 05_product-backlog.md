### สำหรับลูกค้า (Customer)

**การเลือกดูสินค้าและค้นหา**

* **As a customer,** I want to browse products by category so that I can find specific types of caps (e.g., Snapback, Beanie).
* **As a customer,** I want to search for caps by name or keyword so that I can quickly find what I’m looking for.
* **As a customer,** I want to view product details and high-quality images so that I can see the material and design of the cap.

**การจัดการตะกร้าสินค้าและการสั่งซื้อ**

* **As a customer,** I want to add caps to my shopping cart so that I can purchase multiple items at once.
* **As a customer,** I want to adjust quantities or remove items from my cart so that I can finalize my selection before checkout.
* **As a customer,** I want to enter my shipping address and contact info so that I can receive my order.

**การชำระเงินและติดตามสถานะ**

* **As a customer,** I want to pay via multiple methods (e.g., QR Code, Credit Card) so that I can choose the most convenient way.
* **As a customer,** I want to receive an order confirmation email so that I have a record of my purchase.
* **As a customer,** I want to track my order status and shipping number so that I know when my cap will arrive.

---

### เจ้าของร้าน (Admin/Owner)

**การจัดการสต็อกและสินค้า**

* **As an admin,** I want to add, edit, or remove products so that the catalog stays up-to-date.
* **As an admin,** I want to manage stock levels so that I don't oversell products that are out of stock.
* **As an admin,** I want to create categories so that I can organize products effectively.

**การจัดการคำสั่งซื้อและการจัดส่ง**

* **As an admin,** I want to view all incoming orders so that I can prepare items for shipping.
* **As an admin,** I want to update order statuses (e.g., Processing, Shipped) so that customers are informed.
* **As an admin,** I want to verify payment proofs (e.g., slip uploads) so that I can confirm orders for delivery.

**หน้ารายงสนวิเคราะห์ข้อมูล**

* **As an admin,** I want to view a sales report dashboard so that I can see which cap models are bestsellers.

---

### สำหรับระบบสมาชิก (User Management)

* **As a user,** I want to register and log in so that I can save my shipping address for future purchases.
* **As a user,** I want to view my order history so that I can see my past purchases and re-order easily.
